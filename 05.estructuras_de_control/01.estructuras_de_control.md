Las estructuras de control incluyen tanto los condicionales, como los bucles.

## Condicionales

### Si (selección simple)

```yaml
SI (Condición) ENTONCES
  INSTRUCCIONES
FIN DEL SI
```

### Si .... Sino (doble selección)

```yaml
SI (condición) ENTONCES
  INSTRUCCIONES 1
SINO 
  INSTRUCCIONES 2
FIN DEL SI
```

### Según (selección múltiple)

```yaml
SEGÚN (valor)
  valor1 INSTRUCCIÓN1
  valor2 INSTRUCCIÓN2
  valor3 INSTRUCCIÓN3
  ## ...
  valorn INSTRUCCIÓNn
FIN DEL SEGÚN
```

## Bucles

### Mientras

```yaml
MIENTRAS (condición) HACER
  INSTRUCCION
FIN DEL MIENTRAS
```

### Hacer

```yaml
HACER (n veces)
  INSTRUCCIONES
FIN DEL HACER
```

### PAra

```yaml
PARA (i=valor_inicial; i <= valor_final; i++) HACER
  INSTRUCCIONES
FIN DEL PARA
```
